# 抖音小游戏适配指南

## 📋 适配步骤

### 1. 引入抖音小游戏 SDK

在 `index.html` 的 `<head>` 中添加：
```html
<script src="https://lf1-cdn-tos.bytegoofy.com/obj/tos-cn/iqdyd176/minigame/tt-mini-game.js"></script>
```

### 2. 适配触摸事件

抖音使用 `tt.onTouchStart` 和 `tt.onTouchEnd`：

```javascript
// 原代码
canvas.ontouchstart = e => { e.preventDefault(); if (state === 'start') state = 'grow'; };
canvas.ontouchend = e => { e.preventDefault(); if (state === 'grow') state = 'rotate'; };

// 抖音适配
tt.onTouchStart(() => { if (state === 'start') state = 'grow'; });
tt.onTouchEnd(() => { if (state === 'grow') state = 'rotate'; });
```

### 3. 适配本地存储

```javascript
// 原代码
localStorage.setItem('stickBest', best);
const best = localStorage.getItem('stickBest') || 0;

// 抖音适配
tt.setStorage({
    key: 'stickBest',
    data: best.toString(),
    success: () => console.log('存储成功')
});

tt.getStorage({
    key: 'stickBest',
    success: (res) => {
        best = parseInt(res.data) || 0;
    }
});
```

### 4. 适配屏幕尺寸

```javascript
// 获取系统信息
tt.getSystemInfo({
    success: (res) => {
        const screenWidth = res.windowWidth;
        const screenHeight = res.windowHeight;
        // 根据屏幕尺寸调整游戏
        canvas.width = screenWidth;
        canvas.height = screenHeight;
    }
});
```

### 5. 添加广告（可选）

```javascript
// 激励视频广告
let videoAd = tt.createRewardedVideoAd({
    adUnitId: '你的广告位 ID'
});

// 播放广告
videoAd.show().catch(() => {
    videoAd.load().then(() => videoAd.show());
});

// 监听奖励
videoAd.onClose(res => {
    if (res && res.isEnded) {
        // 给予奖励（复活、双倍分数等）
    }
});
```

### 6. 添加分享功能

```javascript
tt.shareAppMessage({
    title: '棍子英雄 - 你能得多少分？',
    imageUrl: 'https://你的域名/cover.jpg'
});
```

## 🚀 提交流程

1. **注册抖音小游戏开发者账号**
   - 访问：https://developer.open-douyin.com/
   - 完成实名认证

2. **创建小游戏应用**
   - 登录开发者后台
   - 创建新应用，填写基本信息
   - 获取 AppID

3. **配置 game.json**
   ```json
   {
       "deviceOrientation": "portrait",
       "showStatusBar": false,
       "networkTimeout": {
           "request": 5000
       }
   }
   ```

4. **上传代码**
   - 下载抖音小游戏开发者工具
   - 导入项目
   - 填写 AppID
   - 上传代码

5. **提交审核**
   - 填写版本信息
   - 提交审核（通常 1-3 个工作日）
   - 审核通过后发布

## ⚠️ 注意事项

1. **包体大小限制**：首包 ≤ 4MB，总包 ≤ 10MB
2. **性能要求**：保持 60fps，避免卡顿
3. **内容规范**：不能含有违规内容
4. **广告规范**：不能强制观看广告
5. **隐私政策**：需要添加用户隐私协议
