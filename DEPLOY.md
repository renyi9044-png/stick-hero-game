### 方式 3：使用内网穿透（推荐）
如果需要外网访问，可以使用以下工具：

1. **ngrok**（推荐）
   ```bash
   ngrok http 8080
   ```
   会生成一个公网 URL，如：`https://xxx.ngrok.io`

2. **Cloudflare Tunnel**
   ```bash
   cloudflared tunnel --url http://localhost:8080
   ```

3. **localtunnel**
   ```bash
   npx localtunnel --port 8080
   ```

## 手机操作说明

1. 打开手机浏览器（Chrome/Safari 等）
2. 输入上述任一地址
3. 游戏会自动适配手机屏幕
4. **操作方式：**
   - 长按屏幕 → 棍子向上生长
   - 松开屏幕 → 棍子向右倒下
   - 搭桥成功后火柴人会自动走过去

## 注意事项

⚠️ **防火墙设置**
如果手机无法访问，可能需要关闭 Windows 防火墙或添加端口例外：
- 端口：`8080`
- 协议：`TCP`

⚠️ **服务器停止**
关闭终端或按 `Ctrl+C` 可停止服务器

## 游戏特性

✅ 响应式设计 - 自动适配手机/平板/电脑屏幕
✅ 触摸支持 - 完美支持手机触摸操作
✅ 横向卷轴 - 流畅的相机跟随效果
✅ 本地存储 - 自动保存历史最高分
