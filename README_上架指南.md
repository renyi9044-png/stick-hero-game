# 🎮 棍子英雄 - 抖音小游戏上架指南

## 📋 准备工作

### 1. 注册抖音开放平台账号

1. 访问：https://developer.open-douyin.com/
2. 注册开发者账号
3. 完成实名认证
4. 缴纳保证金（个人开发者约 1000 元）

---

## 🚀 上架流程

### 第一步：创建小游戏

1. 登录抖音开放平台
2. 进入「小游戏管理」
3. 点击「创建小游戏」
4. 填写基本信息：
   - **名称**：棍子英雄
   - **分类**：游戏 → 休闲益智
   - **图标**：144x144px PNG
   - **描述**：经典休闲小游戏，考验你的判断力！

### 第二步：下载抖音小游戏开发者工具

1. 访问：https://developer.open-douyin.com/devtool
2. 下载并安装「抖音小游戏开发者工具」
3. 用抖音扫码登录

### 第三步：导入项目

1. 打开开发者工具
2. 点击「导入项目」
3. 选择项目目录：`games/douyin/stick-hero/`
4. 填写 AppID（在开放平台创建应用后获取）
5. 点击「导入」

### 第四步：适配抖音小游戏 API

> ⚠️ **重要**：抖音小游戏需要适配其 API 体系

创建 `js/main.js` 替换原生 JS：

```javascript
// 适配抖音小游戏
const tt = tt || {};

// 游戏主逻辑
function init() {
    // ... 游戏初始化代码
}

// 适配触摸事件
tt.onTouchStart((res) => {
    if (state === 'start') state = 'grow';
});

tt.onTouchEnd((res) => {
    if (state === 'grow') state = 'rotate';
});
```

### 第五步：测试

1. 在开发者工具中点击「预览」
2. 扫码在手机上测试
3. 检查：
   - ✅ 触摸操作正常
   - ✅ 显示效果正常
   - ✅ 无报错

### 第六步：提交审核

1. 点击「上传」
2. 填写版本号（如 1.0.0）
3. 填写版本说明
4. 上传后在开放平台提交审核

### 第七步：等待审核

- 审核时间：1-3 个工作日
- 审核通过后自动上线
- 可在后台查看审核进度

## ⚠️ 注意事项

### 1. 内容合规
- ❌ 不能有暴力、血腥内容
- ❌ 不能有赌博元素
- ❌ 不能有诱导分享
- ✅ 休闲益智类最容易过审

### 2. 性能要求
- 首屏加载时间 < 3 秒
- 内存占用 < 200MB
- 帧率稳定在 30fps 以上

### 3. 广告变现（可选）

在 `game.json` 中添加广告配置：

```json
{
  "adConfig": {
    "bannerAd": true,
    "rewardedAd": true,
    "interstitialAd": true
  }
}
```

广告接入代码：

```javascript
// 激励视频广告
tt.createRewardedVideoAd({
    adUnitId: '你的广告单元 ID'
});

// 插屏广告
tt.createInterstitialAd({
    adUnitId: '你的广告单元 ID'
});
```

## 💰 变现模式

### 1. 广告收入
- **Banner 广告**：底部横幅，收益较低
- **激励视频**：看广告复活/双倍分数，收益中等
- **插屏广告**：游戏结束时弹出，收益较高

### 2. 内购（需要版号）
- 购买皮肤
- 购买道具
- 去除广告

### 3. 分成比例
- 抖音平台：50%
- 开发者：50%

## 🔧 常见问题

### Q1: 没有版号能上架吗？
**A**: 可以！小游戏类目不需要版号，但不能有内购，只能广告变现。

### Q2: 审核被拒怎么办？
**A**: 根据拒审理由修改，常见原因：
- 内容违规
- 性能问题
- 隐私政策缺失

### Q3: 如何推广？
**A**: 
- 抖音短视频引流
- 达人试玩
- 买量投放
- 社交分享
